# ZundaTalk

ずんだもんの音声合成・アバター・字幕をWebで連携する簡易システムです。  
OBSのブラウザソースで使用することを想定しています。

## 機能
- VOICEVOXによる音声合成
- WebSocketで音声・字幕・アバターを同期
- 口パク・まばたきアニメーション

## セットアップ
1. `setup.bat`を実行
2. VOICEVOXエンジンを起動
3. `run.bat` を実行
4. `avatar.html` と `subtitle.html` をOBSのブラウザソースで開く
